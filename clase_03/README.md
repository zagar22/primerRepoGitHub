# Git BRANCH


## creo la rama
```sh
git branch dev
```
## git alias

```sh
git config --global alias.lg "log --oneline --decorate --all --graph"
git config --global alias.set "config --global"
git set alias.s "status --short"
```

## para ver los alias
```sh
git config --global --get-regexp alias
git config --global --get-regexp user
```
